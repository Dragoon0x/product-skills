## The Philosophy

Most products fail not because they're built poorly but because they're built strategically — solving the wrong problem, for the wrong audience, at the wrong time, with the wrong metrics.

product-skills encodes the judgment layer: the frameworks, mental models, and decision patterns that determine whether what gets built actually matters. Strategy before features. Discovery before design. Metrics before optimization. Communication before alignment.
